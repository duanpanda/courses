Solution to the knowledge representative bonus problem using semantic web.

Main idea of the solution: symbolic and list computation.

Main features:
1. forward query using function PROP-QUERY which can handle single inheritance;
2. backward query using function REVERSE-PROP-QUERY;
3. collect all the items that have the property with the given value.

Language:      Common Lisp
Language Tool: SBCL
